Dicas para nomear bem variaveis
	Descreva o que a variavel guarda;

	Não use nomes muito curtos ou muito longos;

	Nomes muito curtos somente dentro de escopos pequenos;
	ex: for (int i = 0; i<10 ; i++)

	Evitar nomes invertidos;

	Usar variaveis de tamanho adequado ao tamanho da informação;
	
	tipos de variaveis:
	byte -> -128 - 127;
	short -> -32765 - 32767;
	int -> -2147483648 - 2147483647
	...

	Nomear arrays com pluralidade;
